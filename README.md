#  News Intelligence Dashboard (Web Scraper Project)

A multi-source web scraping and news intelligence dashboard built using Python and Streamlit.  
This project collects real-time news from multiple sources, processes it, and presents insights like trending topics and AI-style summaries.

---

## 🚀 Features

- 🌍 Multi-source news scraping (BBC + Google News RSS)
- 📂 Category filtering (World, Business, Technology, Sports, Politics)
- 🔎 Search functionality for filtering news
- 📅 Proper date parsing and sorting (Latest / Oldest)
- 🔥 Trending topics extraction using keyword analysis
- 🧠 AI-style news summary (extractive summarization)
- 📰 Clean dashboard UI using Streamlit
- 📄 Direct news links for full articles

---

## 🛠️ Tech Stack

- Python
- Streamlit
- Feedparser (RSS scraping)
- Pandas
- TextBlob (sentiment analysis)
- Collections (keyword analysis)

📊 How It Works
Fetches news from multiple RSS feeds
Extracts title, link, and published date
Cleans and processes text data
Performs keyword extraction for trending topics
Generates AI-style summary of headlines
Displays everything in an interactive dashboard
